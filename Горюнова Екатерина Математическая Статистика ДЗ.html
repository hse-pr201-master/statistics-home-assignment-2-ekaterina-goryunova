def get_bootstrap_sample(x, B_sample=1):
    np.random.seed(10)
    N = x.size 
    sample = np.random.choice(x, size=(N, B_sample), replace=True)
    
    if B_sample == 1:
        sample = sample.T[0]
    return sample

# Классический доверительный интервал
def asci(x_mean, sigma = 1, alpha = 0.05):
    left1 = x_mean - sts.norm.ppf(1 - alpha/2) * sigma / np.sqrt(20)
    right1 = x_mean + sts.norm.ppf(1 - alpha/2) * sigma / np.sqrt(20)
    #print('Асимптотический нормальный ДИ: [{:.4f}, {:.4f}]'.format(left, right))
    return left1, right1

Известно, что $E(Y_i) = E(Y_{mean})$ и $E(Y_{mean}) = Y_{mean}$

Вопрос:¶

Среди курьеров проводится соревнование по скоростной доставке заказов. Каждый участник имеет 20 попыток, лучшая из которых (минимальное время) засчитывается в качестве итогового результата. Время, за которое случайно выбранный участник (в пределах одной попытки) доставляет заказ, хорошо описывается экспоненциальным распределением с параметром λ и не зависит ни от результатов предыдущих попыток, ни от результатов других участников. Имеется выборка X1, . . . , X500 из итоговых (лучших) результатов 500 участников.

Оцените вероятность того, что худший результат среди первых 50 участников соревнований окажется меньше 1.

Ответ чата¶

Заметим, что время доставки заказа $X_i$ имеет экспоненциальное распределение с параметром $\lambda$ и функцией распределения $F(x) = 1 - e^{-\lambda x}$.

Тогда, если $X_1, \dots, X_{50}$ - лучшие результаты первых 50 участников, то вероятность того, что худший результат среди них будет меньше 1, равна:

$$P(\min\{X_1, \dots, X_{50}\} < 1) = 1 - P(\min\{X_1, \dots, X_{50}\} \geq 1)$$

Заметим, что $\min\{X_1, \dots, X_{50}\}$ также имеет экспоненциальное распределение с параметром $50\lambda$.

Тогда:

$$P(\min\{X_1, \dots, X_{50}\} \geq 1) = e^{-50\lambda}$$

И, следовательно:

$$P(\min\{X_1, \dots, X_{50}\} < 1) = 1 - e^{-50\lambda}$$

Для оценки вероятности $P(\min\{X_1, \dots, X_{50}\} < 1)$ нужно оценить параметр $\lambda$. Для этого воспользуемся выборкой $X_1, \dots, X_{500}$ и методом максимального правдоподобия.

Функция правдоподобия для выборки $X_1, \dots, X_{500}$ имеет вид:

$$L(\lambda) = \prod_{i=1}^{500} \lambda e^{-\lambda X_i} = \lambda^{500} e^{-\lambda \sum_{i=1}^{500} X_i}$$

Логарифмируя функцию правдоподобия, получаем:

$$\ln L(\lambda) = 500\ln\lambda - \lambda\sum_{i=1}^{500} X_i$$

Дифференцируя логарифм функции правдоподобия по $\lambda$ и приравнивая производную к нулю, получаем оценку параметра $\lambda$:

$$\hat{\lambda} = \frac{500}{\sum_{i=1}^{500} X_i}$$

Теперь можем оценить вероятность $P(\min\{X_1, \dots, X_{50}\} < 1)$:

$$P(\min\{X_1, \dots, X_{50}\} < 1) = 1 - e^{-50\hat{\lambda}} = 1 - e^{-\frac{25}{\bar{X}}}$$

где $\bar{X} = \frac{1}{500}\sum_{i=1}^{500} X_i$ - выборочное среднее.
